# Hear Me
This repository contains the code for the mobile app Hear Me.  
Hear Me is an app developed with React Native to assist differently abled students take exams online with the help of an AI assistant.  

# Stack Used
MERN (MySQL, Express, React Native & Node.js)  

MySQL: Database  
Express: Middleware (API routing)  
React Native: Frontend  
Node.js: Backend processing  

# Code Structure Explained
The src folder has 3 main subfolders.  
The Express folder has the code with all the API routing calls between the frontend and backend (database).  
The Python_Face_ID folder has the code for face recognition login server hosted locally using flask.  
The React_Native folder has the javascript files used to create all the frontend pages and navigation between the pages.  
Note: The database schema is in the img folder.

# How to run
Download and install node.js, npm, flask, MySQL, React Native, Express.js & PyTorch.  
Make sure to download the expo app on an iOS device to test the app.  
Run the combined.py file on a device on the same network as the iOS device.  
Run the app.js file in the Express folder on a device on the same network as the iOS device.  
Run the React Native app on your iOS device using npm run.
